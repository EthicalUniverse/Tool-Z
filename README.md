### $ How to Install

Open the terminal and type following commands.

* `apt update`

* `apt install git`

* `git clone https://github.com/EthicalUniverse/Tool-Z`

* `cd Tool-Z`

* `chmod +x *`

* `sh install` if not work than use `./install`
